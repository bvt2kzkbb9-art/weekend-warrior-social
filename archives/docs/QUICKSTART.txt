1. Otwórz terminal w katalogu projektu:

   https://weekend-warrior-social-ed3d0.web.app

═══════════════════════════════════════════════════════════════════════════════
OPCJA 2: MANUALNE WDROŻENIE (SZCZEGÓŁOWE)
═══════════════════════════════════════════════════════════════════════════════

KROK 1: PRZYGOTOWANIE
─────────────────────

npm install -g firebase-tools
firebase login
firebase use weekend-warrior-social-ed3d0

KROK 2: WALIDACJA
─────────────────

firebase validate

(Powinieneś zobaczyć: ✔ firestore.rules, storage.rules validated successfully)

KROK 3: DEPLOYMENT RULES
────────────────────────

firebase deploy --only firestore:rules

(Czekaj ~30 sekund)

KROK 4: DEPLOYMENT STORAGE
───────────────────────────

firebase deploy --only storage

(Czekaj ~30 sekund)

KROK 5: DEPLOYMENT INDEXES
───────────────────────────

firebase deploy --only firestore:indexes

(Czekaj 5-15 minut! Indeksy są tworzone w tle)

KROK 6: DEPLOYMENT HOSTING
───────────────────────────

firebase deploy --only hosting

(Czekaj ~1-2 minuty)

KROK 7: PROPAGACJA CDN
──────────────────────

Czekaj 2-5 minut na pełną propagację

KROK 8: WERYFIKACJA
────────────────────

Otwórz w przeglądarce:
https://weekend-warrior-social-ed3d0.web.app

TROUBLESHOOTING - ROZWIĄZYWANIE PROBLEMÓW
═══════════════════════════════════════════════════════════════════════════════

❌ Problem: "Permission denied"
✓ Rozwiązanie: firebase logout && firebase login

❌ Problem: "firestore.rules not found"
✓ Rozwiązanie: Sprawdź czy jesteś w katalogu projektu
  cd /home/user/weekend-warrior-social && ls firestore.rules

❌ Problem: "Assets returning 404"
✓ Rozwiązanie: Redeploy hosting
  firebase deploy --only hosting

❌ Problem: "Service Worker not registering"
✓ Rozwiązanie: Hard refresh w przeglądarce (Ctrl+Shift+R)
  Czekaj 2-3 minuty, spróbuj ponownie

❌ Problem: "Firestore Indexes stuck in 'Building'"
✓ Rozwiązanie: To jest normalne!
  Indeksy mogą być budowane 5-15 minut
  Czekaj aż status zmieni się na "Enabled"

❌ Problem: "App returns blank page"
✓ Rozwiązanie: Sprawdź JavaScript errors
  F12 > Console > szukaj czerwonych błędów
  Sprawdź czy manifest.json się ładuje

AKTUALIZACJE & MAINTENANCE
═══════════════════════════════════════════════════════════════════════════════

Po każdej zmianie w kodzie:
1. Zatwierdź zmiany w Git
2. Uruchom deploy.sh lub firebase deploy --only hosting
3. Poczekaj na propagację CDN (2-5 min)
4. Czyszcz cache przeglądarki (Ctrl+Shift+Delete)

Aktualizacja reguł Firestore:
1. Edytuj firestore.rules
2. firebase deploy --only firestore:rules

Aktualizacja reguł Storage:
1. Edytuj storage.rules
2. firebase deploy --only storage
